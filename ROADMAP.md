# chrono ŌĆö Roadmap

> **Version:** 0.1.0 | **Updated:** 2026-06-16
> **Package:** `chrono`

Inventory of every shipped unit, what it provides, and where its verification
artifact lives. The Non-goals and Deferred-refinements tables at the bottom say
what is explicitly out of scope. `Ōśģ` marks the spine (a unit nothing else can
substitute for).

---

## Definition of Done

Every module in the tables below meets all six criteria. The gatekeeper rule
across the whole repository.

| Criterion       | What it requires                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Built from spec | Re-implemented from RFC / ISO / POSIX / IANA ŌĆö never ported, never `sed`-ed from an existing library.                                                                                                                                                                                                                                                                                                                                                            |
| Typed surface   | Compile-time parameters make wrong combinations unrepresentable (`Instant[clock]`, `Timezone[transition_count, type_count]`).                                                                                                                                                                                                                                                                                                                                    |
| Verified        | Per-variant KAT in `tests/` + differential against an independent oracle (Python `datetime` / `zoneinfo` / `dateutil`) at generation time + a seam test in `tests/test_api.mojo` exercising every public re-export.                                                                                                                                                                                                                                              |
| Formatter clean | `mblack` clean, zero compiler warnings, errors raise (`tests/run_tests.mojo` re-raises any `f > 0` so the shell exits non-zero).                                                                                                                                                                                                                                                                                                                                 |
| Naming          | Per [docs/05-naming.md](docs/05-naming.md) ŌĆö full words, purpose-accurate, one concept one name across the whole library.                                                                                                                                                                                                                                                                                                                                        |
| Honesty note    | Every public module declares a `Facets:` line (tier ┬Ę safety ┬Ę quantum) and a `Honesty:` clause stating exactly what the verification covered ŌĆö no overclaim. Pure private-helper modules (`chrono/_internal/*`, `chrono/_core/units.mojo`, `chrono/_core/clock_id.mojo`, `chrono/_core/clock.mojo`) carry only a top-line summary, because they have no independent verification surface ŌĆö their behavior is exercised by the public modules that consume them. |

## Engineering conventions (enforced)

Repository-wide rules that gate every PR. Each is a hard line, not a guideline.

| Convention                  | Detail                                                                                                                                                                                                                                                                                                               |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| No bare ASCII magic numbers | Character bytes are written `ord("T")`, never `== 84`. Named constants in `chrono/_internal/charset.mojo` for the bytes referenced more than once.                                                                                                                                                                   |
| Cursor-based parsers        | Fixed-format timestamps go through `chrono/_internal/scanner.mojo` `Scanner` (`take_int` / `expect` / `accept` / `take_fraction_nanoseconds`), not hand-indexed byte arithmetic.                                                                                                                                     |
| Shared text helpers         | `pad`, `fractional_seconds`, `format_offset_hhmm`, `fold_leap_second` all live in `chrono/_internal/text.mojo` ŌĆö never copied per-module.                                                                                                                                                                            |
| Stdlib over hand-rolling    | `String.split`, `.lower()`, slicing, `InlineArray` literal init `[a, b, c]`. The only hand-written byte loop is the ISO 8601 duration body (`chrono/format/iso_duration.mojo`) ŌĆö its `P{Y,M,W,D}T{H,M,S}` grammar with optional fractional and strict-canonical-order rules doesn't fit any stdlib splitter cleanly. |
| Zero runtime data           | Timezone + country lookup is `comptime`-baked. No file I/O at runtime, no `/usr/share/zoneinfo` dependency, no startup parse cost.                                                                                                                                                                                   |
| Errors raise                | No silent fallback, no return-of-default. Every parser carries the offending value + the valid range in its error message. The test runner re-raises on `f > 0` so a failing suite exits non-zero in the shell.                                                                                                      |
| Idioms documented           | Every Mojo language idiom in use (e.g. `comptime if`, `var self`, `InlineArray` parametrization, `TrivialRegisterPassable`) is enumerated in [docs/07-idioms.md](docs/07-idioms.md) ŌĆö the canonical reference for new code.                                                                                          |

## Non-goals (deliberate)

| Area                          | Decision                                                                                                                                                                                                                                                                                                                                                                                  |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Localization beyond English   | `Strftime` reads month + weekday names from the `Month` / `Weekday` enums; `NaturalDate` reads only weekday names (its month-related grammar uses the `month`/`year` TimeUnit keywords, not month names). No `Locale` type, no CLDR bundle. Future localization would mirror the typed-zone layer as `chrono/locales/<code>.mojo` ŌĆö NOT the runtime glibc-locale loader that was removed. |
| String-keyed timezone lookup  | `TimeZone.load(name)` removed. Mojo's compiler stalls past several hundred arms over distinct parametric `Timezone[N, T]` instantiations (empirically aborted at the 776-arm dispatch). Users import the constant directly (`from chrono.timezones.asia.bangkok import BANGKOK`); backward-compat IANA names re-export through `chrono.timezones.aliases`.                                |
| Leap-aware "right/" timelines | The Unix timeline ignores leap seconds in arithmetic. `:60` is parsed only at `23:59 UTC` (RFC 3339 ┬¦5.6) and folded to `:59`; elsewhere it raises. ASN.1 DER rejects `:60` outright per X.690 ┬¦11.7.                                                                                                                                                                                     |
| Calendar systems beyond three | The `Calendar` trait + Gregorian / Julian / tabular Islamic are the shipped set. Hebrew, Chinese, observational Hijri, and similar are out of scope ŌĆö the trait is open, so a user can conform their own struct to `Calendar` and share the epoch-day pivot without modifying chrono.                                                                                                     |
